## 3. Mimari

```
İstemci: PWA (Next.js 15 App Router + TypeScript + Tailwind + shadcn/ui)
   ├─ pdf.js (PDF reader)
   ├─ IndexedDB (offline cache + queue)
   └─ Service Worker (PWA shell)
        ↓ HTTPS
Vercel Edge (static assets + SSR)
        ↓
Cloudflare Worker (TR endpoint)
   ├─ /ai/chat     → Gemini Flash (free tier) → fallback Haiku
   ├─ /ai/grade    → AI değerlendirme (Pratik Olay)
   ├─ /pdf/:key    → R2 signed URL
   └─ /sync/*      → D1 read/write
        ↓
Cloudflare R2  (590 MB PDF, free tier 10 GB)
Cloudflare D1  (progress, history, plan, ~50 MB beklenen)
```

## 4. Modüller

### Modül 1 — Quiz Engine
- **Veri:** Statik JSON, ~80 soru × 15 ders = ~1.200 MC soru
- **Modlar:** Rastgele · Konu odaklı · Zayıf konu · HMGS Mock (120 soru/4 saat)
- **Soru kaynağı:** Ders özetleri + HMGS geçmiş sınavlar (5 adet)
- **AI:** Yok — tüm sorular ön-üretilmiş
- **Faz:** 2 (HMGS dönemi öncelik)

### Modül 2 — Flashcard (SRS)
- **Veri:** ~150 kart × 15 ders = ~2.250 kart
- **Algoritma:** Basitleştirilmiş SuperMemo 2 (Bildim/Kararsız/Bilmedim)
- **Senkron:** D1 üzerinden Mac↔iPhone progress sync
- **AI:** Yok — kartlar ön-üretilmiş
- **Faz:** 1 sonu (final tekrarı için ezber pekiştirme)

### Modül 3 — PDF Reader + AI Chat ⭐
- **Akış:** PDF aç → paragraf seç → "🤖 Bu ne demek?" / "Pratik örnek" / "Özet"
- **RAG:** Önceden hesaplanmış embedding'ler + browser-side cosine similarity
- **Embeddings:** OpenAI text-embedding-3-small (tek seferlik, ~$0.50)
- **AI:** Gemini Flash → 5K input + 1K output ≈ $0.003/soru
- **Faz:** 1 (finallere acil yardım)

### Modül 4 — Plan & Dashboard
- **Plan:** HMGS tarihinden geri sayım, ders dağılımı, zayıf konu ağırlığı
- **Dashboard:** Streak, haftalık total, ders pie chart, Pomodoro entegre
- **AI:** Hafif (haftalık özet için, opsiyonel)
- **Faz:** 2 başı (uzun vadeli planlama önemli olduğunda)

### Modül 5 — Pratik Olay Çözücü
- **Veri:** Yüce Miras Pratik + Özbek Ceza Muhakemesi Pratik + HMGS geçmiş + diğer kaynaklardan çıkarılan ~30-50 olay
- **Akış:** Olay oku → çözümünü yaz → AI karşılaştır → eksikleri göster
- **AI:** Olay başı ~1 çağrı (~$0.01)
- **Faz:** 1 (final pratiği için)

## 5. Veri Modeli

### Statik bundle (frontend ile birlikte)
```typescript
courses: { id, name, topics: [...] }[]
quizzes: { [course_id]: Question[] }
flashcards: { [course_id]: Flashcard[] }
practice_cases: { [course_id]: Case[] }
embeddings: { chunks: Chunk[], vectors: Float32Array }  // ~30 MB
```

### D1 (sync edilen kullanıcı state)
```sql
quiz_attempts(id, question_id, answer, correct, ts)
flashcard_state(card_id, ease, next_review, last_seen, streak)
chat_history(id, course, messages, created_at)
study_plan(date, course, target_min, actual_min, completed)
pomodoro_sessions(id, course, duration, completed_at)
```

### R2 bucket: `hukuk-pdf`
- `dersler/{course}/{filename}.pdf` — orijinal PDF'ler (590 MB)
- `embeddings/{course}/chunks.json` — chunk metadata (ihtiyaç olursa)

## 6. AI Stratejisi

| Use case | Model | Token/req | Maliyet |
|----------|-------|-----------|---------|
| PDF chat (Modül 3) | Gemini Flash (free → Haiku fallback) | ~5K + 1K | $0.003 |
| Olay değerlendirme (Modül 5) | Gemini Flash → Haiku | ~3K + 800 | $0.002 |
| Haftalık özet (Modül 4) | Gemini Flash | ~2K + 500 | $0.001 |

**Optimizasyonlar:**
- Prompt caching (Anthropic): aynı PDF chunk'ı tekrar tekrar gönderilirken %90 indirim
- RAG: tüm PDF değil, semantic search ile ilgili 3-5 chunk
- Rate limit: 100 req/gün/kullanıcı (Worker'da)
- Response cache: aynı soru 3 dk içinde tekrarlanırsa cache hit

**4 ay toplam tahmin:** $0-25 (1 kullanıcı, Gemini free tier yeterli)

## 7. Güvenlik

- API key'ler **sadece** Worker environment'ta (`wrangler secret put`)
- Kullanıcı asla key görmez
- R2 bucket private + Worker signed URL ile serve eder
- Worker rate limit: kullanıcı başı 100 AI çağrısı/gün
- HTTPS only, security headers (Vercel default)
